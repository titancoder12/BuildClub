## Course 1: Build a Web Application

The first Build Club course introduces students to building a complete web application from scratch.

Students learn:
- How the web works
- HTML and CSS for structure and design
- Python and Flask for backend development
- SQLite for data storage
- How AI can be used to build a full web application from the ground up

The course concludes with a live demonstration showing how weeks of manual work can be recreated in minutes using AI â€” reinforcing the importance of understanding fundamentals in the AI era.
