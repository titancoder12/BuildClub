# Admin Guide

## Sign in
1. Visit `/admin/` on your Build Club site.
2. Enter the admin username and password created with `python manage.py createsuperuser`.

## Manage courses
- Open **Courses**.
- Create or edit a course.
- Set `published` to show it on the public site.
- Update location, day/time, session length, and capacity as needed.

## Manage registrations
- Open **Registrations** to review submissions.
- Update the registration status to Pending, Confirmed, Waitlisted, or Cancelled.

## Export registrations
1. In **Registrations**, select the rows to export.
2. Choose **Export selected registrations as CSV** from the action menu.
3. Download the generated CSV file.
